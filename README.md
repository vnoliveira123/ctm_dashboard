# Log Dashboard

Dashboard para análise de processos Control-M com volumetria massiva (50M+ logs).

## Features
- 📊 Tela 1: Processos Cadastrados (tabela filtrada)
- 📈 Tela 2: Detalhes de Processamento (timeline + logs)
- 🔗 Tela 3: Fluxo de Processamento (grafo interativo)

## Quick Start
\\\ash
docker-compose up
\\\

Acesse http://localhost:3000

- Python 3.11 + FastAPI
- React 18 + TypeScript
- PostgreSQL + Redis
- Docker
